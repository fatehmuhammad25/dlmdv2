About This LMS

Project Overview

This Learning Management System (LMS) project was created by Fateh Muhammad as a final assignment for the course Digital Learning Media Development. The goal is to provide a simple, accessible, and interactive platform for delivering learning materials using digital media.

The LMS is built using HTML, CSS, and JavaScript without a server backend, making it lightweight and easy to deploy via platforms like GitHub Pages. It applies four major learning theories: Vygotsky's Social Constructivism, Piaget's Cognitive Constructivism, Sweller's Cognitive Load Theory, and Mayer's Multimedia Learning Theory.

Main features include digital learning materials for each theory, assessments to test understanding, a feedback form to collect responses from users, and a dynamic feedback banner displayed on the homepage.
