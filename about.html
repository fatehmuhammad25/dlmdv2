About This LMS

Project Overview

This Learning Management System (LMS) was designed and developed by Fateh Muhammad as a final assignment for the course Digital Learning Media Development. The aim is to deliver an engaging, accessible, and interactive digital platform for learning key educational theories.

System Architecture & Tools

The LMS was built using pure HTML, CSS, and JavaScript, ensuring it remains lightweight and deployable on free hosting services like GitHub Pages. No backend server is required, but it uses Firebase Realtime Database for real-time data storage and synchronization.

Learning Theories Included

This LMS integrates four core educational theories to guide the learning content:

  1. Vygotsky's Social Constructivism
  2. Piaget's Cognitive Constructivism
  3. Sweller's Cognitive Load Theory
  4. Mayer's Multimedia Learning Theory
